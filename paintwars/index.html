Paintwars (Lua)

Imagine playing a round of Overwatch on a control map. Now, picture all the Overwatch weapons swapped with Splatoon weapons. Finally, imagine the whole game being made in Roblox. That's Paintwars.

  • Create data structure to efficiently store and traverse all blocks in the level.
  • Design and implement eight different weapons.
  • Create game rules.
  • Run playtest sessions.
