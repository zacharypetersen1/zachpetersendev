Flexible Cel Shader (C#, Cg)

Flexible Cel Shader is an Asset Pack I created that is sold on the Unity Asset Store. In the first 6 months of sales it had about $500 of revenue.

  • Write shader with meaningful parameters for artists.
  • Create preset system with JSON so that users can toggle between different styles of the shader with one click.
  • Write custom material editor window to make it easier for users to get the look they want.
  • Create two variations of the shader that use multiple passes to draw outlines.
