Nathair Puzzle Design Tool (Python)

A tool used to generate puzzles for "Nathair". The user defines a set of constraints for the puzzle: where teleporters go, where the puzzle starts, and where there must be a floor or wall tile. The tool then generates all puzzles that satisfy the user-defined constraints.

  • Write search algorithm to generate puzzles.
  • Create user-friendly GUI.
  • Write events that update constraints based on user's actions.
