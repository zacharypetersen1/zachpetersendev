Picar.io (Python)

Picar.io is proof-of-concept multiplayer game for the Pico-8 virtual console based on Agar.io. It is the first large multiplayer game to be made for the Pico-8, capable of supporting 64+ simultaneous players. I wrote the server for this project.

  • Built Python server that updates clients and deployed it on Google Compute Engine.
  • Implemented spatial hashing, reducing data needed to update clients per tick.
  • Implemented seamless add/drop player functionality
